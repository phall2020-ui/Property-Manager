# Screenshots

This directory will contain screenshots of the Appointments and Attachments features.

## Required Screenshots

### Appointments
- [ ] Contractor proposing appointment form
- [ ] Landlord/Tenant confirmation banner
- [ ] Confirmed appointment card
- [ ] Mini calendar with upcoming appointments
- [ ] iCal download action

### Attachments
- [ ] File upload drag-and-drop area
- [ ] Upload progress indicators
- [ ] Image gallery grid view
- [ ] Document list view
- [ ] Image lightbox with zoom/rotate
- [ ] Category tabs (All/Before/After)

## How to Add Screenshots

1. Run the application locally or in a test environment
2. Navigate to a ticket detail page
3. Take screenshots of the features in action
4. Name files descriptively (e.g., `appointment-propose-form.png`)
5. Add screenshots to this directory
6. Update this README with links to images

## Format

- Format: PNG or JPG
- Resolution: 1920x1080 or higher
- Include browser UI for context
